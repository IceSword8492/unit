# Unit


## 準備


### コンパイル


```sh
g++ ./main.cpp -lwinmm
```


## 実行


### 実行コマンド


```sh
./a.exe ([option]...)
```


### オプション


fpsを表示
```sh
-fps
```


2fpsで実行
```sh
-vlfps
```


10fpsで実行
```sh
-lfps
```


60fpsで実行
```sh
-hfps
```


120fpsで実行
```sh
-vhfps
```


240fpsで実行
```sh
-vvhfps
```


教授のスポーン確率を0に変更 (Scoreは0に固定されます)
```sh
-noEnemy
```
